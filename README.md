# template-javascript-basic

javascript研修で使用するテンプレート

## ディレクトリ構成

```
template-javascript-basic/
├── .github/                        # GitHub に関する設定ファイル
│   └── pull_request_template.md    # PR 作成時に自動で入力されるテンプレート
├── .vscode/                        # VSCode エディタの設定フォルダ。チーム全員が同じ環境で開発できるよう設定を共有する
│   ├── extensions.json             # このプロジェクトで使うべき VSCode 拡張機能のリスト。開いたとき「インストールしますか？」と表示されるもの
│   └── settings.json               # 保存時の自動整形・インデント幅など、エディタの動作をプロジェクト専用に設定するファイル
├── .gitignore                      # Git の管理対象から除外するファイルやフォルダを指定するファイル。「.DS_Store」など不要なものをコミットしないために使う
├── index.html                      # ブラウザで最初に開く HTML ファイル。ページの骨組み（見出し・テキスト・ボタンなど）をここに書く
├── style.css                       # デザイン・見た目を定義するスタイルシート
├── script.js                       # 動作・インタラクションを実装する JavaScript ファイル
└── README.md                       # このファイル。プロジェクトの説明書
```

## コード規約

- シングルクォートを使用する
- セミコロンを付ける
- インデントは2スペース
- 不要な変数を作らない
- == ではなく === を使用する
